# Python Programming Examples - Jupyter Notebooks

This directory contains organized Jupyter notebooks with Python programming examples, complete with explanations and implementation details.

## Available Notebooks

### 1. example_set_1_basic_programs.ipynb
**Content**: Basic Python programming concepts
- Rectangle area calculation
- Simple interest calculation  
- Temperature conversion (Celsius to Fahrenheit)
- Average of three numbers
- Square and cube calculation
- Number swapping techniques
- Student report generation

**Learning Objectives**:
- Basic input/output operations
- Arithmetic operations and mathematical formulas
- Variable manipulation and data types
- String formatting and output display

### 2. example_set_2_loops_conditionals.ipynb
**Content**: Control structures and iteration
- Sum of first 10 natural numbers
- Factorial calculation
- Fibonacci series generation
- Finding largest among three numbers
- Complete student result system with grading

**Learning Objectives**:
- For loops and range function
- Conditional statements (if-elif-else)
- Accumulator patterns
- Grade assignment logic
- User input validation

### 3. example_set_3_functions_classes.ipynb
**Content**: Advanced programming concepts
- Function definitions and implementations
- Natural numbers printing
- Number reversal algorithms
- Digit counting functions
- Palindrome checking
- Fibonacci series using functions
- Calculator with basic operations
- File operations (read/write)
- Exception handling (division by zero)
- Object-oriented programming with Student class

**Learning Objectives**:
- Function definition and calling
- Parameter passing and return values
- Error handling with try-except blocks
- File I/O operations
- Class definition and object creation
- Method implementation
- Modular programming design

## How to Use

### Running the Notebooks
1. Open Jupyter Notebook or JupyterLab
2. Navigate to the notebooks directory
3. Open the desired notebook file
4. Run cells individually or all at once
5. Follow the interactive prompts for input

### Interactive Learning
- Each notebook cell contains executable code
- Comments explain the implementation approach
- Output is displayed immediately after execution
- Modify code to experiment with variations
- Add new cells to test different scenarios
