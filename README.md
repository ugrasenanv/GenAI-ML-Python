#### Example 8: Sum of Natural Numbers
- **Implementation**: Loop-based accumulation with step-by-step display
- **Algorithm**: Initialize sum=0, iterate 1 to n, accumulate results
- **Concepts**: For loops, range function, accumulator patterns
- **Applications**: Mathematical series, statistical calculations

- **Implementation**: Sequence generation using variable swapping technique
- **Pattern**: Each number = sum of previous two numbers
- **Concepts**: Series generation, state management, mathematical sequences
